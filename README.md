# NotePad Pro (ipadwritting)

A high-performance, iPad-optimized note-taking web application using React, TypeScript, and the HTML5 Canvas API.

## Features
- Multi-page document system
- Apple Pencil support (Canvas API)
- Pen, Highlighter, Eraser, Lasso, and Shape tools
- Draggable and resizable Text and Image elements
- Local storage persistence with IndexedDB
- Voice recording directly onto notes
- PDF Annotation and export
- Dark mode support
- Folders and Context menus

## Setup
```bash
npm install
npm run dev
```
